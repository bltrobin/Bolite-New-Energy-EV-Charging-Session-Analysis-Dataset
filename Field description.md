# BLT Dataset Field Descriptions / 字段解释

## Battery Charging Parameters / 电池充电参数

| Field / 字段 | Chinese / 中文 | English / 英文 |
|-------------|---------------|----------------|
| `time` | 充电时间 | Charging Time |
| `a` | 额定容量 | Rated Capacity |
| `b` | 此次充电容量 | Current Charging Capacity |
| `c` | BMS实时电流 | BMS Real-time Current |
| `c1` | 快充结束电流 | Fast Charging End Current |
| `c2` | 慢充开始电流 | Slow Charging Start Current |
| `d` | BMS记录时间 | BMS Recording Time |
| `e` | BMS实时电压 | BMS Real-time Voltage |
| `e1` | 快充结束电压 | Fast Charging End Voltage |
| `e2` | 慢充开始电压 | Slow Charging Start Voltage |
| `f` | 现存容量百分比 | Current Capacity Percentage |
| `f1` | 电池内部电阻 | Battery Internal Resistance |
| `g` | 此次充电能量 | Current Charging Energy |
| `h` | BMS实时充电功率 | BMS Real-time Charging Power |
| `h1` | BMS平均充电功率 | BMS Average Charging Power |
| `i` | 额定电池电压 | Rated Battery Voltage |
| `j` | 现存电池能量 | Current Battery Energy |
| `k` | 能量保持率 | Energy Retention Rate |
| `l` | 上次充电能量 | Last Charging Energy |
| `m` | 电池循环次数 | Battery Cycle Count |
| `m1` | 距离上次电池循环差值 | Difference from Last Battery Cycle |
| `n` | 额定电池能量 | Rated Battery Energy |
| `o` | SOC开始 | SOC Start |
| `p` | SOC结束 | SOC End |
| `q` | SOC差 | SOC Difference |
| `q1` | 最大温差SOC | Maximum Temperature Difference SOC |
| `r` | 此次衰败每循环 | Current Degradation per Cycle |
| `r1` | 平均衰败每循环 | Average Degradation per Cycle |
| `s` | 充电开始温度 | Charging Start Temperature |
| `t` | 充电结束温度 | Charging End Temperature |
| `u` | 升温/1%充电 | Temperature Rise per 1% Charge |
| `u1` | 归一升温/1%充电 | Normalized Temperature Rise per 1% Charge |
| `w` | 电池种类 | Battery Type |
| `x` | 温度差可接受值 | Acceptable Temperature Difference |
| `z` | voltage vs. time负曲率 | Voltage vs. Time Negative Curvature |
